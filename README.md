## Descripción
Este taller consiste en la implementación de 14 programas en Java que resuelven problemas clásicos de programación utilizando funciones recursivas. 

## Instrucciones para ejecutar
Para ejecutar cualquier ejercicio, abra una terminal en la raíz del proyecto y ejecute los siguientes comandos reemplazando N por el número del ejercicio que desea probar (del 1 al 14).

Compilar:
javac src/Recursividad/EjercicioN.java

Ejecutar:
java -cp src Recursividad.EjercicioN

El programa solicitará los datos de entrada por consola.
